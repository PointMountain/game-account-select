## 分类原则

```yaml
asset:
  name: string
  type: core_character|equipment|resource|progression|account_risk|unknown
  account_value: high|medium|low|unknown
  notes: string
```

## 高价值资产类别

- 命名核心角色/装备/账号资产。
- 与用户目标一致的队伍、养成或资源组合。
- 可验证的抽卡、货币、材料或进度资源。

## 低价值或易误判资产类别

- 只写总数，不列名称。
- 高等级但资源和核心资产缺失。
- 账号绑定、实名、找回状态不明。

## 待刷新

- 当前版本核心资产 tier。
- 当前版本资源和账号市场价值。
- 账号绑定和找回风险规则。
