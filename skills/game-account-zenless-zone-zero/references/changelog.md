# 绝区零（ZZZ）skill changelog

## 2026-05-24

- 明确绝区零资源折算口径：`菲林` 和 `菲林底片` 都按 160:1 折算，`加密母带` / `原装母带` / `邦布券` 按 1:1 折算，避免把 `菲林底片` 误算成单抽券。
- 根据人工复盘修正性价比排序：三虚狩 `2+1` 优先级高于耀嘉音/耀佳音 `1+1`，耀嘉音 `0+0` 可用；非虚狩角色通常 `0+1` 优先于 `1+0`；妄想天使三小只最好都有专武，南宫羽专武为最低硬优先。
- 增加 `JHYXJ3297` 正例和耀嘉音 `1+1` 低投入反例，验证低价三虚狩 `2+1`、三小只专武和直伤电完整度能正确压过单一耀嘉音舒适度。

## 2026-05-23

- 根据用户筛号反馈修正三虚狩队伍硬口径：星见雅看柚叶+南宫/狼/苍角，叶瞬光看照+耀嘉音/琉音，仪玄看卢西娅+橘福福/琉音。
- 增加直伤电队与异放队检查：直伤电按希希芙+席德/希德+耀嘉音，异放队按妄想天使三小只千夏+爱芮+南宫羽。
- 增加薇薇安紊乱队舒适度加分：薇薇安 + 明确紊乱/异常搭档可作为额外队伍价值信号。
- 增加舒适度加分规则和回归验证：虚狩2+1、额外队伍核心0+1、耀嘉音/耀佳音1+1均应影响排序，但不能替代硬条件。
- 增加螃蟹/盼之详情页角色角标规则：优先使用 `agentStatuses` 判断影画；专武用 `x+y` 或 `S级音擎` 名称清单 + 本地 `signature-engines.json` 交叉确认。
- 新增本地专武数据库 `references/signature-engines.json`，覆盖三虚狩、关键辅助、直伤电、妄想天使、薇薇安紊乱队和常见常驻 S 代理人的专属音擎名称。
- 将真实买号筛选的社区证据刷新窗口从 30 天收紧到 7 天。
- 增加邮箱交付/实名风险规则：邮箱未实名出售作为低找回风险加分项，邮箱不出售和邮箱状态不明进入高风险确认。
- 增加邮箱风险回归验证，确保同等资产下邮箱未实名出售优先于邮箱不出售。
- 基于三虚狩买号复盘补充社群证据：星见雅优先查柚叶，叶瞬光查琉音/照/千夏/耀嘉音，仪玄查琉音/橘福福/青衣与卢西娅/潘引壶。
- 增加“三虚狩独立三队”硬规则：同一辅助不能被多个虚狩重复计算，三虚狩齐但无法拆成三队的账号只能中低置信备选。
- 增加三队完整性回归验证，确保“3 虚狩 + 2 共享辅助”的低价账号不会排在三队完整账号前。
