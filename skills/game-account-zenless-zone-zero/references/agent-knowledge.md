## 常驻 S / 易误判资产

```yaml
standard_or_lower_priority:
  - Nekomata
  - Soldier 11
  - Koleda
  - Lycaon
  - Rina
  - Grace
```

常驻 S 不是无价值，但不能因为高影画或数量多就压过限定核心队伍。

## 队伍判断

- 主 C/异常核心 + 击破/支援/防护 + 合适邦布/音擎，队伍价值高。
- 限定代理人带专属音擎明显加分。
- 孤立限定但无队友、无音擎、无资源，只能中等评分。
- 常驻高影画只有在能补齐强队定位时才加中低权重。
