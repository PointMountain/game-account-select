# Neverness to Everness（异环）资产知识表

## 分类原则

```yaml
asset:
  name: string
  type: s_character|s_arc_plate|resource|progression|account_type
  account_value: high|medium|low|unknown
  notes: string
```

## 当前稳定资产类别

因为角色和弧盘强度随版本/测试环境变化，本文件当前只维护资产类别，不维护具体角色强度榜。

### 高价值资产类别

- 命名 S 角色，且与用户目标一致。
- 命名 S 弧盘，且明确适配已有 S 角色。
- 关键觉醒节点和高练度核心角色。
- 环石、异晶、质实骰子、三重钥匙等可规划资源。
- 完整截图/文本说明账号类型、绑定、主角选择、区服。

### 低价值或易误判资产类别

- 只写 S 数量，不列名称。
- 弧盘数量多但不列套装/属性/适配角色。
- 猎人等级高但缺少资源和核心资产。
- TAP/完美账号/B服状态不明。
- 主角性别或账号类型与用户偏好冲突。
