# Neverness to Everness（异环）估值规则

## 评分维度

```text
账号价值 = S角色质量 + S弧盘质量 + 觉醒/练度 + 抽卡资源 + 进度便利 - 风险扣分 - 数据缺失扣分
```

## 高权重字段

- S 级角色数量和质量
- S 级弧盘数量和适配度
- 关键角色觉醒
- 环石、异晶、质实骰子、三重钥匙
- 账号是否符合用户要求的主角和区服/账号类型

## 低权重字段

- 单纯总黄数但缺少角色/弧盘明细
- 猎人等级但资源和核心角色不足

## 风险扣分

- TAP 已绑定
- 账号类型不符合用户要求
- 描述缺少 S 角色、S 弧盘或资源细节
- 平台保障不足
