# Wuthering Waves（鸣潮）估值规则

updated_at: 2026-05-16

## 最高优先级规则

不要把总黄数、五星角色数量、五星武器数量当作主价值来源。它们只能作为粗略资产规模参考。

必须先判断高命角色属于：

1. 当前版本强势限定
2. 可组成关键队伍的限定角色
3. 有专武支撑的核心输出/辅助
4. 常驻或弱势角色

常驻/弱势角色高命只给低权重，不能因为命座高就显著提高推荐排名。

## 社区证据要求

评分前必须读取 `references/community-evidence.md`，或使用调用方传入的同结构证据快照。

若快照满足以下任一情况，`confidence` 最高只能为 `medium`：

- `updated_at` 距当前日期超过 30 天。
- 游戏已进入新大版本或卡池环境显著变化。
- 账号核心角色未出现在快照中。
- B站/小红书/抖音等中文社区来源全部不可用，只能依赖英文榜单。

若社区证据和账号描述冲突，优先降低置信度，不要用模型常识补齐。

## 评分维度

```text
最终Wuthering Waves（鸣潮）资产分 = 限定强度分 + 热门配队/队伍完整度分 + 主C专武适配分 + 资源分 - 风险扣分 - 数据缺失扣分
```

## 可执行评分框架

总分建议使用 100 分上限，最后按风险与缺失扣分得到排序分。

```yaml
score_weights:
  limited_core_score: 0-35
  team_archetype_score: 0-10
  team_completeness_score: 0-15
  signature_weapon_score: 0-15
  resource_score: 0-15
  price_fit_score: 0-10
  collection_bonus: 0-5
  risk_penalty: 0-35
  missing_data_penalty: 0-25
```

### 限定核心分

- 高价值限定核心：每个 8-10 分。适用多来源高位或当前版本强势角色，如 Hiyuki、Aemeath、Lynae、Mornye、Augusta、Cartethyia、Shorekeeper、Galbrena、Qiuyuan、Phrolova、Iuno。
- 中价值限定/常用角色：每个 4-6 分。适用 Jinhsi、Zani、Phoebe、Carlotta、Camellya、Changli、Jiyan、Verina、Cantarella、Zhezhi、Yinlin 等仍有用途但不是本快照最高共识的角色。
- 常驻、过时或弱势角色：每个 0-2 分。高命也不得突破此区间，除非社区证据明确显示其在当前版本关键队伍中有高价值。

### 队伍完整度分

队伍分由“热门配队识别”和“可用队伍数量”组成。不要只看命座或五星武器数量。

### 热门配队/队伍原型分

社媒热门配队、深塔/海墟高频队和当前版本攻略中反复出现的队伍方向，应作为独立价值来源。示例：

- 爱莫林：Aemeath + Mornye + Lynae。命中 2 人给 4-6 分，命中 3 人给 8-10 分；主 C 或关键位带专武时再走专武分。
- 卡千夏：Cartethyia + Chisa + Xia Kong / Aero 相关队友。命中 Cartethyia 且队友明确给 5-8 分，队伍与专武完整给 8-10 分。
- 日月守：Jinhsi / 日系主 C + 月相关辅助 + Shorekeeper。因社区命名可能漂移，命中明确队伍关系时给 4-8 分，缺队友截图时只给低段。
- Augusta + Iuno：命中二者且角色定位清楚给 6-9 分。
- Hiyuki / 绯雪相关队：若有当前版本适配队友和专武，给 5-9 分；只有孤立绯雪不满分。

这些队伍名是购买决策标签，不是永久强度定论。若社区证据超过 30 天或新版本改变环境，必须降低置信度并建议刷新。

### 可用队伍数量分

- 能组成一队当前高价值完整队：8-11 分。
- 能组成两队深塔/海墟/高难可用队：12-15 分。
- 只有孤立强角色、缺关键辅助或缺武器：3-7 分。
- 只有角色数量但队伍关系不明：0-3 分。

### 专武与命座分

- 高价值主 C 或队伍关键位带专武：每个 5-6 分。
- 中价值角色带适配 5 星武器：每个 2-3 分。
- 常驻/弱势角色高命或专武：每项 0-1 分。
- 命座只有在角色本身属于高价值核心，且社区证据确认该命座质变时，才给 3-6 分。否则只给 0-2 分。

主 C 是否有专武必须和角色/队伍关系一起看：

- 主 C 高命但无专武、队友缺失：命座分不应满给。
- 主 C 低命但有专武且队伍完整：可高于孤立高链号。
- 专武数量很多但归属不明：先扣数据缺失，不把“五星武器数”直接转成高分。

### 资源分

高价值资源包括：

- 星声
- 月相
- 浮金波纹
- 铸潮波纹
- 余波珊瑚

资源多适合追求后续抽卡和版本适应性的用户。

- 可支撑一个版本抽卡规划：10-15 分。
- 有明显余量但不足以覆盖新限定规划：5-9 分。
- 资源少但已有完整队伍：1-4 分。
- 资源未披露：0 分并加缺失扣分。

### 价格适配分

价格必须结合用户预算和风险偏好，不使用绝对市场价拍脑袋。

- 预算内且资产命中用户目标：7-10 分。
- 预算内但缺关键字段：3-6 分。
- 超预算或用低价值资产堆高价格：0-2 分。

### 收藏加分

皮肤、绝版纪念、特殊收藏最多 5 分。收藏偏好不是用户核心目标时，不得让收藏资产压过强度/风险。

## 风险

重点扣分：

- 已绑定 TAP
- 已绑定 Wegame
- PS5/第三方账号关联状态不明
- 换绑冷却
- B服/渠道服不符合用户要求
- 无找回包赔
- 无官方验号
- 描述中缺少角色命座、武器或绑定状态

风险扣分建议：

- 已绑定且不可换绑、找回风险高：20-35 分。
- 换绑 CD、PS5/TAP/Wegame 状态不明：8-18 分。
- 已明确未绑定 Wegame/防沉迷无，但 TAP/PS5/邮箱换绑仍未截图确认：4-10 分，并列入人工确认。
- 无找回包赔或官方验号：8-15 分。
- 渠道服不符合用户要求：直接硬筛失败，除非用户确认可接受。
- 异常低价且资产描述过好：10-25 分并要求人工确认。

## 数据缺失扣分

缺失字段不能当作好消息：

- 不列角色命座：扣 8-15 分。
- 不列专武或武器：扣 5-10 分。
- 不列资源：扣 4-8 分。
- 不列绑定/换绑/找回保障：扣 10-20 分。
- 只给截图但 OCR 不完整：扣 5-12 分并把置信度降到 `low|medium`。

## 排序硬规则

- “多常驻高命 + 少量限定 + 无专武/绑定不明”的账号，不得排在“少量但当前高价值限定核心 + 专武/资源/风险明确”的账号前面。
- 只写总黄数、五星数量或“极品号”但没有资产明细的账号，不能进入 Top 1。
- 孤立高命主 C 不得自动排在热门配队完整、主 C 有专武、风险明确的账号前面。
- 资源号只能作为“未来抽卡弹性”推荐；用户目标是强度/成品时，不得压过明确成型队伍。
- 找回/绑定风险高的账号，即使资产强，也必须在推荐理由里置顶风险；用户偏好低风险时应排除。
- 社区证据未覆盖的新角色、新命座、新武器，不能直接按高价值处理。

## 推荐解释要求

每个推荐账号必须明确说明：

- 值钱的角色/命座是否是限定或强势角色
- 是否命中当前热门配队或至少形成可解释队伍
- 主 C 是否带专武，专武归属是否明确
- 常驻高命是否被低权重处理
- 抽卡资源是否足够
- 绑定风险是否可接受
- 哪些字段需要人工确认
- 本账号相对社区证据快照是“强一致”“部分一致”“弱一致”还是“冲突”
