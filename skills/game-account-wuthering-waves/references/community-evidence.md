## 共识归纳

### 高价值资产

- 当前版本强势或多榜单高位限定：Hiyuki、Aemeath、Lynae、Mornye、Augusta、Cartethyia、Shorekeeper、Galbrena、Qiuyuan、Phrolova、Iuno。
- 能形成完整队伍的组合资产比单个孤立角色更值钱，例如 Aemeath + Lynae + Mornye，Hiyuki + 其关键队友规划，Augusta + Iuno，Cartethyia 相关 Aero 队。
- 核心限定角色带专武或明确可用 5 星武器时，价值明显高于只有角色但无武器/资源的账号。

### 中价值资产

- 仍能打但多榜单不在最高位或榜单差异较大的限定/常用角色：Jinhsi、Zani、Phoebe、Carlotta、Camellya、Changli、Jiyan、Verina、Cantarella、Zhezhi、Yinlin 等。
- 辅助/副 C 价值取决于已有主 C 和队伍完整度；不能只按稀有度加分。

### 低价值或陷阱资产

- 常驻或早期角色高命不能直接当作高价理由，尤其是 Lingyang、Calcharo、Jianxin、Encore 等缺少当前限定队伍支撑时。
- 只写“总黄数”“五星很多”“满命”但不列角色、命座、专武和绑定状态的账号，应降低置信度并扣数据完整度。
- B站买号/避坑搜索结果集中出现找回风险、换绑 CD、PS5 关联账号等主题，因此绑定状态和找回包赔必须作为硬风险项。

## 账号估值落地规则

1. 先识别高价值限定核心和队伍完整度，再看五星数量。
2. 专武与关键命座只在对应角色是当前高价值或队伍核心时给高权重。
3. 常驻/弱势高命最多给低权重，不能让账号越过低链强限定队。
4. 资源（星声、月相、浮金波纹、铸潮波纹、余波珊瑚）用于衡量后续抽卡弹性；缺资源的“成品号”要看是否已形成至少两队。
5. 绑定、换绑、PS5/TAP/Wegame、渠道服、找回包赔和官方验号缺失时，优先扣风险，不用资产分掩盖。
